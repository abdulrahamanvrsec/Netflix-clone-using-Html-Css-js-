# 🎬 Netflix Clone - Frontend UI

A clean and fully responsive **Netflix homepage clone** built using only **HTML** and **CSS**. This project replicates Netflix’s landing page design with static layout and modern styling.

---

## 🚀 Features

- 📱 Fully responsive layout (Mobile, Tablet, Desktop)
- 🎨 Pure HTML and CSS – no frameworks or JS
- 🖼️ Realistic Netflix homepage UI: navbar, banner, content rows
- 💡 Hover effects, font styling, and layout design
- ⚡ Fast-loading and lightweight

---

## 🛠️ Built With

- **HTML5** – Semantic page structure
- **CSS3** – Styling and responsive layout (Flexbox/Grid)
